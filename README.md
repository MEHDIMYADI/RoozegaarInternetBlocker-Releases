# Roozegaar Internet Blocker

Roozegaar Internet Blocker is a lightweight Windows application built with WinUI 3 that provides a simple and reliable way to control internet access for desktop applications.

Unlike many similar tools, Roozegaar Internet Blocker works entirely through native Windows Firewall rules. It does not require a database, background service, or kernel driver, making it fast, secure, and resource-efficient.

## Features
- Lightweight and modern WinUI 3 interface
- Block or restore internet access for any Windows application
- Uses native Windows Firewall rules
- No database required
- No background service or Windows service
- Enable or disable firewall rules with a single click
- Easily add or remove executable files
- Available in both Installer and Portable editions
- Minimal CPU and memory usage
- Supports Windows 10 and Windows 11

# اینترنت بلاکر روزگار

Roozegaar Internet Blocker یک نرم‌افزار سبک برای ویندوز است که با WinUI 3 توسعه یافته و امکان مدیریت دسترسی اینترنت برنامه‌های ویندوز را به ساده‌ترین شکل فراهم می‌کند.

برخلاف بسیاری از نرم‌افزارهای مشابه، این برنامه تنها از قوانین داخلی Windows Firewall استفاده می‌کند و برای عملکرد خود به دیتابیس، سرویس ویندوز یا درایور کرنل وابسته نیست؛ در نتیجه سریع، ایمن و کم‌مصرف است.

## ویژگی‌ها
- رابط کاربری مدرن و روان مبتنی بر WinUI 3
- مسدود یا آزاد کردن دسترسی اینترنت هر برنامه
- استفاده مستقیم از Windows Firewall
- بدون نیاز به دیتابیس
- بدون نیاز به Windows Service
- فعال یا غیرفعال کردن قوانین تنها با یک کلیک
- افزودن و حذف آسان فایل‌های اجرایی (EXE)
- ارائه در دو نسخه نصبی و پرتابل
- مصرف بسیار کم حافظه و پردازنده
- سازگار با Windows 10 و Windows 11
